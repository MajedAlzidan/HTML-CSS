# HTML-CSS

## Bootstrap Movie Project
- Created a movie showcase website using Bootstrap.
- Features navbar bookmarks, jumbotron, movie cards, pill badges for showtimes, and a footer.

## Portfolio Website
- Built a colorful and unique portfolio using HTML and CSS.
- Includes:
  - Fixed navbar
  - Centered image with CSS filter and rounded edges
  - Styled table with borders, padding, and colored header
  - Text overlay on images
  - Two-column layout
  - Hover effects for text and images
  - Image overlay fade effect
  - Embedded autoplay/muted/loop video with text overlay
  - Styled footer

Both projects demonstrate responsive design, creativity, and modern web development practices.
